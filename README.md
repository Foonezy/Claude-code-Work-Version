## О проекте

<div align="center">

*Иногда вещи появляются там, где их не ждали.*

</div>

<br>

Это — **скомпилированная версия Claude Code**, которая обычно остаётся внутри корпоративной инфраструктуры.

Где-то что-то пошло не так.  
Где-то что-то оказалось не там, где должно было быть.  

И в какой-то момент появился готовый результат.

Без сборки.  
Без зависимостей.  
Без долгих инструкций.

Просто рабочий инструмент.

<br>

<div align="center">

**Ирония в том, что сложные вещи иногда приходят в максимально простом виде.**

</div>
