## О проекте

Иногда технологии выглядят сложными.  
Многоступенчатая сборка, зависимости, конфигурации, окружение.

А иногда — всё происходит совсем иначе.

Без предупреждений.  
Без лишнего шума.  
Без подготовки.

Просто в какой-то момент появляется уже готовая версия.

Та самая, которую обычно никто не планировал распространять.  
Та самая, где всё уже собрано за тебя.

И в этом есть что-то почти комичное.

Потому что весь сложный процесс… внезапно исчезает.
