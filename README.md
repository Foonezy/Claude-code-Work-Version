# Claude Code (Compiled)

Рабочая скомпилированная версия Claude Code.

---

## Описание

Ситуация получилась максимально классическая:

Кто-то где-то что-то «не туда нажал»,  
кто-то «не заметил»,  
и в итоге появилась версия, которую обычно никто не должен видеть.

Вместо долгих инструкций по сборке, зависимостей и танцев с окружением —  
просто готовый билд.

Ирония в том, что иногда самые сложные вещи приходят в максимально простом виде.
